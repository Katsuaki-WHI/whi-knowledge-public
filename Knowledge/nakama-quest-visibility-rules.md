# なかまクエスト：視認性デザインルール

## 背景色 × 文字色の組み合わせ

| 背景 | 文字色 | 用途 |
|---|---|---|
| 暗い背景（青グラデ） | 白 または 金色 `#FFE566` | メイン表示 |
| 明るい背景（白/クリーム`rgba(255,253,240,0.95)`） | 濃い茶色 `#180800` | カード・本文 |
| クリーム背景の選択肢カード | 濃い文字 `#180800` + 金色枠線 `2px solid #E8A000` | 選択肢 |

## GoldButton特有のルール

- 文字色（`color`）は **`#180800`** （単色）
- グラデーション文字列を`color`に指定すると無効値となりブラウザフォールバックバグ発生
- → 必ず単色を指定する
