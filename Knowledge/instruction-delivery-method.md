# Claude Codeへの指示書：サイズで渡し方を使い分ける

## 原則

指示書のサイズに応じて、渡し方を使い分ける。

## 使い分け基準

| サイズ | 行数の目安 | 渡し方 |
|---|---|---|
| 短い | 100行・3KB以下 | チャット内コードブロックで直貼り |
| 中 | 100〜500行・3〜15KB | どちらでもOK。かつさんの好みに合わせる |
| 大 | 500行以上・15KB以上 | ファイル経由（ダウンロード→パス取得→Claude Code） |

## ファイル経由の手順

1. claude.ai から指示書をダウンロード
2. ~/Downloads/ に保存（Macなら通常の保存先）
3. Optionキー+右クリック → 「パス名をコピー」でフルパス取得
4. Claude Codeに「[フルパス] を読んで実行してください」と渡す

## コードブロック直貼りの手順

1. claude.ai のチャットでコードブロック内の指示書全文を提示
2. かつさんがコピー
3. Claude Codeのプロンプトにペースト

## 切り替えるときのルール

やり方を変えるときは、必ず理由と新旧の違いを明示する。
「急にやり方が変わった」状態を避ける。
