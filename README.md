# WHI Knowledge Base (Public Mirror)

This is a public mirror of WHI's knowledge base, intended for claude.ai web_fetch access.
Sensitive content (pricing, infrastructure secrets, project status) is kept in the private repository.

## Purpose

Enable claude.ai (web/desktop/mobile) to read foundational philosophy, principles, and conversation rules
via `web_fetch` at the start of new conversations.

## Usage

At the start of a new chat:

```
Please read https://raw.githubusercontent.com/Katsuaki-WHI/whi-knowledge-public/main/INDEX.md
and https://raw.githubusercontent.com/Katsuaki-WHI/whi-knowledge-public/main/Knowledge/mistakes.md
before we begin.
```
